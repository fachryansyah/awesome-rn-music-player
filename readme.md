## How to install
###1. Clone repo
cloning code to your local computer<br>

```$ git clone ```

###2. Install prerequiste library
Install of prerequiste library for the project

```$ npm install```

###3. Run to your android
```$ npm run android```

### Or to your iOS
```$ npm run ios```
